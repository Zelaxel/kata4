# Kata4
Continuación de la kata 4. Implementación de un JFrame para mostrar un grafico.
## Objetivos
- **Diseño del Model**: clases que representan los registros.
- **Diseño del View Model**: histograma
- **Diseño del View**: código encargado de generar el gráfico.
- **Diseño del Control**: código que conecta ambas partes.
## Implementación
Planteamos la clase MainFrame que gestiona la interfaz gráfica.
